# Twitter Backend
## Platform Used
### Backend :
- Java (version : JDK 21) <br>
  The Twitter Backend is Developed With Java (JDK 21). Java is a versatile and widely-used programming language known for its portability, platform independence, and robustness. Originally developed by Sun Microsystems and released in 1995, Java has since become one of the most popular programming languages in the world.
  <br><br>
- MySQL (version : 8.0.35) <br>
  The Twitter Bcakend is made by the MySQL (8.0.35) database Language. MySQL is a widely-used open-source relational database management system (RDBMS) that is known for its reliability, performance, and ease of
  use. It has become one of the most popular database systems in the world and is frequently used in web development for powering dynamic websites and applications.
  <br><br>
### Installation :
#### JAVA :
To Install Java in Windows, Go to The Official JAVA Website by [Click Here](https://www.java.com/download/ie_manual.jsp) and download the latest java Package (current : JDK21).
If you Don't Know the Complete Installation Process, then Watch this [Youtube Video](https://youtu.be/BB0gZFpukJU?si=3btCfglWoAGMEQPR).
#### MySQL :
To Install MySql in Windows, Visit The The Official MySQL site by [Click Here](https://dev.mysql.com/downloads/installer/) and Choose the Latest Supported version (current : 8.0.35) click the MSI installer and Download it. the follow the steps By Watching This [Youtube Video](https://youtu.be/s6DQHBdMpvs?si=ENaQpLU1u1qG00fR).
